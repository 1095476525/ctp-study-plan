◆ OnFrontDisconnected()

virtual void CThostFtdcMdSpi::OnFrontDisconnected ( int  nReason)
inlinevirtual

This function is invoked when the connection is ended or disconnected.

If the message is left unprocessed, then the API instance will automatically reconnect with server using one of the front addresses from the registered front addresses list.

Parameters
[out]nReasonError reason, should be one of:
Code(Hex) Code(Dec) Description
0x1001 4097 Network reading failed.
0x1002 4098 Network writing failed.
0x2001 8193 Heartbeat receving timeout.
0x2002 8194 Heartbeat sending timeout.
0x2003 8195 Received an error message.

◆ OnRtnDepthMarketData()

virtual void CThostFtdcMdSpi::OnRtnDepthMarketData ( CThostFtdcDepthMarketDataField *  pDepthMarketData)
inlinevirtual

Server uses this callback function to notify the latest market infomation of subscribed instrument.

Parameters
[out]pDepthMarketDataA pointer to an CThostFtdcDepthMarketDataField instance.
Note
CThostFtdcDepthMarketDataField.TradingDay will be updated around 17:30 (based on New York Time) per day.
The update time of the quotation is splitted into 3 fields: CThostFtdcDepthMarketDataField.ActionDay, CThostFtdcDepthMarketDataField.UpdateTime, and CThostFtdcDepthMarketDataField.UpdateMillisec.
For example:
ActionDay UpdateTime UpdateMillisec Aggregated Update Time in China Standard Time
"20200702" "05:00:00" 0 "2020-07-02 05:00:00.000"
"20200702" "10:05:00" 18 "2020-07-02 10:05:00.018"
"20200702" "15:10:05" 999 "2020-07-02 15:10:05.999"
See also
CThostFtdcMdApi::SubscribeMarketData()
