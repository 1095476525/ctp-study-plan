Introduction of API files

File Name Description Note
ThostFtdcTraderApi.h C++ header file of trading interface.
ThostFtdcMdApi.h C++ header file of quotation interface.
ThostFtdcUserApiStruct.h Defines the data structures.
ThostFtdcUserApiDataType.h Defines the data types.
thosttraderapi.dll The dynamic library of trading interface. Windows only.
thosttraderapi.lib The import library of thosttraderapi.dll. Windows only.
thostmduserapi.dll The dynamic library of quotation interface. Windows only.
thostmduserapi.lib The import library of thostmduserapi.dll. Windows only.
libthosttraderapi.so The dynamic library of trading interface. Linux only(Need to handle SIGPIPE signal).
libthostmduserapi.so The dynamic library of quotation interface. Linux only(Need to handle SIGPIPE signal).
error.xml Defines the api error code and information.
error.dtd The document type definition of error.xml.
Note
For MS VC 6.0, MS VC .NET 2003 users, please turn on the multi-thread option in the compile settings.
For Linux users, the client program may receive SIGPIPE signal triggered by the operating system. By default, the client program will be terminated by operating system when received SIGPIPE, client need to change the default behaviour of handling SIGPIPE signal. Please refer to the Q&A section for details.
