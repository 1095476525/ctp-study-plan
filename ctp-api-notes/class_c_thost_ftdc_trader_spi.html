◆ OnFrontDisconnected()

virtual void CThostFtdcTraderSpi::OnFrontDisconnected ( int  nReason)
inlinevirtual

This function is invoked when the connection is ended or disconnected.

If the message is left unprocessed, then the API instance will automatically reconnect with server using one of the front addresses from the registered front addresses list.

Parameters
[out]nReasonError reason, should be one of:
Code(Hex) Code(Dec) Description
0x1001 4097 Network reading failed.
0x1002 4098 Network writing failed.
0x2001 8193 Heartbeat receving timeout.
0x2002 8194 Heartbeat sending timeout.
0x2003 8195 Received an error message.

◆ OnRspQryInvestorPosition()

virtual void CThostFtdcTraderSpi::OnRspQryInvestorPosition ( CThostFtdcInvestorPositionField *  pInvestorPosition,
CThostFtdcRspInfoField *  pRspInfo,
int  nRequestID,
bool  bIsLast 
)
inlinevirtual

Response result of query investor position request.

Parameters
[out]pInvestorPositionA pointer of an CThostFtdcInvestorPositionField instance.
[out]pRspInfoA pointer to the system response include the status code.
[out]nRequestIDRequest id of the corresponding request.
[out]bIsLastIndicate if it is the last response of the same request.
Note
A positive value of position means Long and a negative value means Short.
CThostFtdcInvestorPositionField.PositionDate is always set to THOST_FTDC_PSD_Today.
User can get the current net position via CThostFtdcInvestorPositionField.Position, today's net position via CThostFtdcInvestorPositionField.TodayPosition, and yesterday's net position via CThostFtdcInvestorPositionField.YdPosition.
There are two types of position: uni-directional position and bi-directional position.
For uni-directional position, position of the same instrument should be open only in one direction, Long or Short.
Therefore, only net position will be returned, the field CThostFtdcInvestorPositionField.PosiDirection is aways set to THOST_FTDC_PD_Net.
For bi-directional position, Long and Short position of the same instrument can be kept at the same time.
Therefore, both Long position and Short position will be returned, the field CThostFtdcInvestorPositionField.PosiDirection is set to THOST_FTDC_PD_Long for Long position; the field CThostFtdcInvestorPositionField.PosiDirection is set to THOST_FTDC_PD_Short for Short position.
By default, if there are bi-directional positions, they will be converted to uni-directional positions for display, to be compatible with older versions.
Please contact support team for further assistance if you want to display bi-directional positions.
See also
CThostFtdcTraderApi::ReqQryInvestorPosition()
