# CTP API 学习笔记

简介
- 本仓库用于整理和维护对 CTP API 的学习笔记、示例与检查点（checkpoints）。

目录结构（初始）
- ctp-api-notes/
  - README.md                  ← 此文件
  - checkpoints/
    - 000-initial-summary.md    ← 第一份会话摘要与学习计划
  - examples/
    - .gitkeep                  ← 占位，后续存放示例代码

使用说明
- 每次重要会话或里程碑我会生成一个 checkpoints/NNN-*.md 文件作为“检查点”摘要，包含问题清单、关键结论与下一步行动。
- 我会在每次新增内容时做增量摘要（delta），并在必要时合并历史检查点以节省 token。
- 提交信息示例（用于自动化或手动提交）："Add initial CTP API learning notes and first checkpoint"

作者与协作
- 我（Assistant / Copilot）将负责生成与更新笔记内容；你负责审核与合并 PR（如采用新分支工作流）。
